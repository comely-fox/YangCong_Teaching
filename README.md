# vue-yc

## Step1. 首先

```
npm install --global tsc
npm install
npm run build
```

## Step2. 其次
```
cd server
npm install
```

## Step3. 然后
```
在src/entry的config.ts文件中配置 user的name与password属性，配置成自己的洋葱账号，可以从官网上注册获取
```

## Step4. 最后
```
打开浏览器： 输入 http:localhost:3000, 下载自己需要的视频即可。
下载的视频：默认存放于 server/videos目录下
```
